## Cómo jugar

|   | **Teclado**  | 
| ------ | ------ | 
| **Movimiento** | WASD | 
| **Camara** | Ratón | 
| **Correr** | Shift Izq. | 
| **Ataque** | Click izq.  | 
| **Ataque especial** | Click der. | 
| **Inventario** | Q | 
| **Interacción** | E | 
| **Pausa** | Esc |
